# Sentinel: The Proactive Life-Saving Ecosystem

## Overview
Sentinel is a next-generation emergency response mobile application designed to protect users when they cannot protect themselves. While traditional apps require manual interaction, Sentinel uses **AI-driven environmental awareness** to detect danger and trigger life-saving protocols automatically.

## Key "Sellable" Features
- **AI Acoustic Analysis:** Detects car crashes, screams, and distress phrases in real-time.
- **Stealth Mode (Ghost Protocol):** Triggers silent alerts, audio recording, and GPS tracking without illuminating the screen.
- **Biometric Monitoring:** Syncs with HealthKit/Google Fit to detect sudden physiological distress (heart failure, seizures).
- **Mesh-Network SOS:** Relays emergency signals via Bluetooth/Wi-Fi Direct in areas without cellular coverage.
- **Dynamic Safe-Route:** Real-time crime data integration to suggest the safest walking paths.

## Tech Stack
- **Frontend:** React Native (Expo)
- **State Management:** Redux Toolkit
- **Backend:** Node.js / Firebase (Real-time DB & Cloud Functions)
- **AI/ML:** TensorFlow Lite (on-device audio classification)
- **Location:** Expo Location + Background Geofencing
